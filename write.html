사용 순서
  1. 폼에 심의 완료 원고를 입력하고 「목록에 추가」를 누릅니다. (여러 개 반복 가능)
  2. 다 넣었으면 맨 아래 「사이트 파일 ZIP 다운로드」를 누릅니다.
  3. 내려받은 ZIP의 압축을 풀고, 그 안의 파일들을 GitHub 저장소에 통째로 올려(덮어쓰기) 주면 1~2분 뒤 반영됩니다.

1 글 작성

기존에 쓴 분류가 자동으로 추천됩니다. 새 분류는 그냥 입력하면 됩니다.
영문으로 지정하면 주소가 post-children-caregiver-benefit.html 처럼 깔끔해집니다. 비워두면 자동 번호가 붙습니다. 한번 정한 주소는 나중에 바꾸지 마세요(검색 색인이 초기화됩니다).
복사한 글은 서식 없이 붙여넣어집니다. 필요한 서식은 위 버튼으로 지정하세요. 첫 이미지가 공유 썸네일(og:image)로 쓰입니다.

2 게시글 목록

    아직 추가된 글이 없습니다.

    3 사이트 파일 내보내기

    ZIP 안에는 board.html, 글 페이지들(post-*.html), sitemap.xml, robots.txt, posts.json이 들어 있습니다. 압축을 풀고 GitHub 저장소에 통째로 올려 덮어쓰면 됩니다. (gen.js는 처음 한 번만 올리면 계속 유지됩니다.)
